<strong>Solo Project: Raspberry Pi Pico-based Reactive Music LED</strong><br><br>

| Part | Description |
| --- | --- |
| Development Board | Raspberry Pi Pico |
| Code Editor | Thonny IDE |
| Bootloader | MicroPython UF2 |

| Packages | • machine (default)<br>• utime (default) |
